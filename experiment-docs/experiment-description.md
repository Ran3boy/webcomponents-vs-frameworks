# Описание эксперимента

Эксперимент сравнивает Web Components, React, Vue.js, Svelte и Lit на одном компоненте «Каталог элементов».

Каждая реализация использует единый датасет `data/dataset-500.json` и поддерживает одинаковые функции:

- отображение карточек;
- поиск;
- фильтрация по категории;
- сортировка;
- сброс фильтров.

Результат эксперимента не сводится к одному победителю. Матрица K1-K10 показывает профиль технологии для разных проектных условий.

## Цепочка доказательств

```text
исходный код -> сценарий проверки -> raw JSON -> processed CSV/JSON -> диаграммы -> scoring justification -> GitHub Pages сайт
```

## Связь с разделом 3.1 ВКР

Раздел 3.1 задает критерии K1-K10 и шкалу 0-3. Этот репозиторий показывает, как шкала применяется к наблюдаемым признакам и измерениям: DOM-инкапсуляции, изоляции CSS, переносимости, tooling, производительности и совместимости с UI-экосистемой.
