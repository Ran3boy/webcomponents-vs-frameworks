# Публикация на GitHub Pages

1. Создать репозиторий на GitHub.
2. Загрузить файлы проекта.
3. Перейти в `Settings`.
4. Открыть `Pages`.
5. В `Source` выбрать `Deploy from a branch`.
6. Выбрать ветку `main`.
7. Выбрать папку `/docs`.
8. Нажать `Save`.
9. Дождаться генерации ссылки.

Перед публикацией рекомендуется выполнить:

```bash
npm install
npm run test:all
```

После этого сайт в `docs/` будет использовать актуальные processed данные.
